# 🎬 模板一·拓展：Slidev 课件脚本安排设计（逐页脚本表）

> **使用场景**：在 `012 教材草稿.md` 已经把一节课 5 个模块（x.0 复习 / x.1 引入 / x.2 Bug 猎人 / x.3 创造 / x.4 复盘）写好之后，
> **进入 Slidev 制作前**的**最后一步规划**。本模板把"课的内容"翻译成"Slidev 一页一页该怎么呈现"。
>
> **核心目标**：在写 Slidev Markdown 之前，**先把每张幻灯片**的（页面布局、版式选择、文字内容、插图内容、音频内容、视频内容、代码块、动画、转场、旁白、用时、备注）**全部敲定**，
> 避免边写边想、边渲染边返工。
>
> **所属章节**：第 ____ 章 第 ____ 节
> **课时类型**：□ 📘 知识讲解课　□ 🐢 海龟绘图课　□ 📝 考级练习课　□ 🔄 章节复习课
> **课时长度**：120 分钟（2 小时）
> **本课涉及的知识点编号**：0001、0002、0003 ……
> **配套 012 教材草稿**：`BOOK/NN 章节名/草稿 012 第 X 课_标题.md`
> **目标 Slidev 产物**：`slides/第X课-标题/slides.md`（命名见 016 模板）

## 🎭 跨页一致性原则（脚本阶段就锁死）

> 这部分在写脚本阶段就要**一次定稿**，否则后面 60 页会越写越乱。

### 1. 主人公 / 视角

- [ ] 全课**只**用 `M-Boy 冒险者` 或 `玩家` 作主语（不要"我们"、"你们"）。
- [ ] 海龟课**只**用 `小海龟助手` 作为 Turtle 的拟人称呼。

### 2. 变量命名

- [ ] 第一节出现的 `backpack` / `diamonds` / `score` 等**沿用**整节课，不要中途换 `inventory` / `gold` / `points`。
- [ ] Bug 修复代码中**故意**保留学生易错的拼写（如 `lootbox` 不要纠正为 `loot_box`），便于后期对照学生作业。

### 3. 主题场景

- [ ] 全课**沿用同一个 Minetest 主线**（如"魔法背包"、"地下城挖矿"、"丛林探险"）。
- [ ] 章节引入页 / 通关仪式页**呼应同一事件**（如 5.1 引入是"村长要战利品报告"，5.4 通关也是"独立写报告"）。

### 4. 课件语言

- [ ] 旁白用 `第二人称`（"你"），不要混用"老师"、"小明"等。
- [ ] 学生提问的引用统一用 `> 💬 学生：` 开头。

### 5. 代码块风格

- [ ] **行内注释**：`# 这是注释`（中文，与项目其它课一致）。
- [ ] **折叠答案**：用 `> ```python ... ```>` 嵌套引用块，避免学生一眼看到答案。
- [ ] **高亮关键行**：用 ` ```python {1,3-4|all} ` 这种 Slidev 语法，按"先讲 1 行 → 再展开"。

## 🔗 与其他模板的协作

```
011 知识点草稿（按"5 步"准备每个知识点）
   ↓
012 教材草稿（按 x.0~x.4 五模块整合成完整 2 小时课）
   ↓
015 Slidev 脚本安排设计（你在这里 —— 把课拆成 40~60 页逐页脚本）
   ↓
016 Slidev 演示文件设计（写真正能被 Slidev 渲染的 slides.md）
   ↓
《第 X 课：标题（知识点）》正式 Slidev 课件
```

### 角色分工

- **011** = "按知识点拆开写" —— 一节课 3~5 个知识单元。
- **012** = "按课次统一组装" —— 把 011 合并为 2 小时课的 5 个模块。
- **015** = "按页拆分课件" —— 把 012 的 5 个模块拆成 40~60 张 Slidev 页面，**锁定每页的版式 / 动画 / 用时 / 旁白**。
- **016** = "按脚本渲染成 Slidev Markdown" —— 严格按 015 写 `slides.md`，是 Slidev 直接消费的成品。

### 015 → 016 的字段对应关系

| 015 脚本表的字段 | 016 slides.md 的对应位置 |
| :--- | :--- |
| 页号 + 版式 | frontmatter 的 `layout:` 指令 |
| 模块（A~H） | `---` 分隔符分段 |
| 核心内容 | 页面主标题 + 正文 markdown |
| 关键元素 | 列表、表格、`<Tweet/>` / `<Tweet>` 等组件 |
| 动画 | `v-click` 指令、`v-motion`、行高亮 `{1,3|all}` |
| 旁白 | 老师口播用，**不进** slides.md，**仅**015 自留 |
| 用时 | 016 的"时间戳"备注（用 `<!-- 60s -->` 形式） |
| 备注 | 备品 / 互动形式 / 板书要求 |
