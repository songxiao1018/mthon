## 1.1 你好呀，世界！

### 背景介绍

你出生在 Minetest 的大草原，身上只有 3 个钻石。你按 `T` 键说话，但系统听不懂中文。你必须学会 Python 语言，才能告诉系统你的财富！

### Multi-TRIX百科

- `print()` = 喊话器：可以让你在屏幕上显示任何你想说的话。
  > 使用方法：`print(内容)` 例如：`print("你想说的话")` 这样就可以了！
- 变量 = 背包：可以用来存储你拥有的东西，比如钻石、金币等。
  > 使用方法：`变量名 = 值` 例如：`diamonds = 3` 就是把数字 3 存到变量 `diamonds` 里。
- 注释 = 说明书：可以用来解释你的代码，让别人更容易理解。
  > 使用方法：单行注释用 `#`，多行注释用 `''' '''`。

### 第一次运行

案例 1：输出字符串

```python
# 输出字符串
print("Hello, World!")
```

案例 2：变量与变量赋值

```python
# 定义一个变量，存储钻石的数量
diamonds = 3
# 输出变量的值
print(diamonds)
```

案例 3：注释的使用

```python
# 这是一个单行注释，解释下面的代码
print("Hello, World!")  # 这也是一个单行注释，解释这一行代码

'''
这是一个多行注释，可以用于解释多行代码
或者提供更详细的说明
'''
print("Welcome to Minetest!")
```


## 1.2 Bug猎人 老师报错了！你能帮老师修复代码吗？

| 错误代码        | 报错信息 (让学生读)              | 学生修正   |
| :-------------- | :------------------------------- | :--------- |
| `print(Hello)`  | `SyntaxError: invalid syntax`    | 加引号     |
| `1diamonds = 3` | `SyntaxError: invalid syntax`    | 改名字     |
| `print（“Hi"）` | `SyntaxError: invalid character` | 改英文括号 |


### print() 常见错误

```python
# 错误示例：忘记引号
print(Hello, World!)

# 错误示例：引号不匹配
print("Hello, World!)
print("Hello, World!')
print('Hello, World!")
print(Hello, World!')

# 错误示例：忘记括号
print "Hello, World!"

# 错误示例：用了中文符号
print（“你好”）
```

### 变量常见错误

```python
# 错误示例：变量名不能以数字开头
1diamonds = 3

# 错误示例：变量名不能包含空格
my diamonds = 3

# 错误示例：变量名不能使用特殊字符
my-diamonds = 3

# 错误示例：变量名不能使用 Python 关键字
class = 3

# 错误示例：变量名区分大小写
diamonds = 3
print(Diamonds)  # 这会导致错误，因为变量名是 diamonds 而不是 Diamonds
```

你加入了一个 Minetest 服务器，打算和小伙伴们一起玩，不过在此之前，你需要向大家介绍一下你自己。请使用 `print()` 输出你的名字、年龄和喜欢的 Minetest 物品。

### 进阶任务 建造基地告示牌

目标：完成一个稍微复杂的综合任务，引入“变量更新”。


*   **任务**：用代码打印一个好看的基地介绍牌。
*   **要求**：必须使用 **3 个变量** 和 **5 个 print 语句**。

> 案例输出：
```
====================
欢迎来到 Multi-Boy的秘密基地
基地等级： 10
防御设施： 钻石墙
====================
```

> 参考代码：

```python
# === 基地信息 ===
base_name = "Multi-Boy的秘密基地"
level = 10
defense = "钻石墙"

# === 打印告示 ===
print("====================")
print("欢迎来到", base_name)
print("基地等级：", level)
print("防御设施：", defense)
print("====================")
```
