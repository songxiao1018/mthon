# 第 2 课：召唤绘图海龟（导入、对象与方法）

## 2.1 章节引入与知识点介绍

### 背景介绍
你成功地在 Minetest 的编程世界中安顿下来。今天，你在探索岛屿时，发现了一片奇怪的海滩，沙地上布满了复杂的几何图案。一只戴着眼镜、背着画板的老海龟缓缓爬过来，它说：“年轻的冒险家，我是指引者‘图图’。这些图案是通往宝藏的地图碎片，但只有懂得‘指挥之术’的人，才能让我复现它们，拼出完整的地图。你准备好学习如何‘召唤’并‘命令’你的绘图伙伴了吗？”

### Multi-TRIX百科
本节课，你将学习如何召唤并指挥一个名为“海龟”的绘图伙伴。

- **导入库 (import)** = 打开工具包：从Python的“百宝箱”里取出一个包含特定功能的工具包，让我们可以使用里面的工具。
  > 使用方法：`import turtle` 或 `import turtle as t`。
  > 示例：`import turtle` 意思是“我要使用画图的工具包”。

- **创建对象 (Object Creation)** = 召唤助手：从工具包里创造一个具体的、可以干活的“小助手”或“工具”。
  > 使用方法：`工具包名.创建函数()`，比如 `turtle.Turtle()`。
  > 示例：`my_pen = turtle.Turtle()` 意思是“用画图工具包里的海龟模具，创造一个叫‘我的笔’的海龟助手”。

- **调用方法 (Method Call)** = 下达命令：向你创造的“小助手”发出具体的行动指令。
  > 使用方法：`助手名.方法名(参数)`，比如 `my_pen.forward(100)`。
  > 示例：`my_pen.left(90)` 意思是“命令‘我的笔’向左转90度”。

### 第一次运行
让我们亲自体验一下召唤和命令海龟的完整过程吧！请运行以下代码，看看会发生什么。

```python
# 示例1：召唤海龟并画一条线
import turtle                     # 1. 打开“海龟绘图”工具包
my_turtle = turtle.Turtle()       # 2. 用工具包里的模具，创造一个名叫my_turtle的海龟助手
my_turtle.forward(100)           # 3. 命令my_turtle助手：向前走100步
turtle.done()                    # 让绘图窗口保持打开
```

```python
# 示例2：使用别名，并绘制一个正方形
import turtle as t               # 1. 打开工具包，并给它起个短名字叫‘t’
pen = t.Turtle()                 # 2. 用‘t’（代表turtle）工具包，创建一个叫pen的海龟助手
pen.forward(100)                 # 3. 命令pen助手向前走100步（画第一条边）
pen.left(90)                     # 命令它向左转90度
pen.forward(100)                 # 画第二条边
pen.left(90)
pen.forward(100)                 # 画第三条边
pen.left(90)
pen.forward(100)                 # 画第四条边，正方形完成！
t.done()
```

```python
# 示例3：用另一个名字创建海龟，并画个三角形
import turtle                     # 1. 导入工具包
painter = turtle.Pen()           # 2. 注意：也可以用.Pen()来创建，名字叫painter
painter.forward(100)
painter.left(120)                # 三角形内角是60度，所以外转角是120度
painter.forward(100)
painter.left(120)
painter.forward(100)
painter.left(120)
turtle.done()
```

## 2.2 Bug猎人：老师报错了！你能帮老师修复代码吗？

图图海龟年纪大了，有时会把指令写错。快来帮它修正代码，找回地图碎片吧！

**常见错误类型：**
1.  **工具包未找到**：忘记“打开工具包”（导入库），就想用里面的工具。
2.  **助手不认识**：给助手起了名字，但下命令时写错了名字。
3.  **命令拼写错误**：给助手下的命令（方法名）拼写错了。
4.  **逻辑小迷糊**：命令的顺序或参数不对，画不出想要的图形。

### 代码修复家

#### 任务1：工具包去哪了？
**任务描述**：下面的代码想画一条线，但运行时报错 `NameError: name 'turtle' is not defined`。它忘记了最重要的一步，请修复它。
```python
# 错误的代码：

```
**报错信息**：`NameError: name 'turtle' is not defined` (名称错误：‘turtle’这个名字没有被定义)
> **提示**：Python不认识`turtle`是什么，因为我们还没有告诉它我们要用这个工具。

#### 任务2：调皮的助手名
**任务描述**：老师创建了一个叫`my_pen`的海龟，但却对着`my_pan`下命令。请根据报错信息修复它。
```python
# 错误的代码：
import turtle
my_pen = turtle.Turtle()  # 海龟的名字是 my_pen
my_pan.forward(100)       # 哎呀，命令错对象了！
```
**报错信息**：`NameError: name 'my_pan' is not defined`

#### 任务3：拼写错误的命令
**任务描述**：`forward`（前进）被拼写错了，海龟听不懂这个命令。请修复它。
```python
# 错误的代码：

t = turtle.Turtle()
t.forword(50)  # 正确的命令是 forward，不是 forword
```
**报错信息**：`AttributeError: 'Turtle' object has no attribute 'forword'` (属性错误：‘Turtle’对象没有‘forword’这个属性/命令)

#### 任务4：混乱的建造顺序
**任务描述**：图图想画一个边长为80的等边三角形，但代码的角度似乎不对，画出来的不是闭合的三角形。请你计算并修正转向的角度。
```python
# 有逻辑错误的代码：
import turtle
tur = turtle.Turtle()
tur.forward(80)
tur.left(60)   # 等边三角形每个内角是60度，但海龟转角是外角，应该是多少度呢？
tur.forward(80)
tur.left(60)
tur.forward(80)
tur.left(60)
turtle.done()
```
> **提示**：海龟转的是外角。对于等边三角形，每次应该转120度（180-60=120）。
