## 绘图任务
**[画面]** 
我们要用海龟画一个彩色的正方形迷宫入口！海龟会从屏幕中心出发，画出一个边长可变的正方形，就像 Minecraft 里的迷宫入口一样。

## 坐标规划
**[数学]** 
- 起点：(0, 0) - 屏幕中心
- 前进 100 像素：从 (0,0) 到 (100,0)
- 右转 90 度：朝向改为向上
- 前进 100 像素：从 (100,0) 到 (100,100)
- 重复 4 次，画出一个正方形

## 核心代码
**[海龟代码]** 

```python
# 导入海龟模块
import turtle

# 创建海龟对象
t = turtle.Turtle()

# 设置速度（1-10，10最快）
t.speed(5)

# 定义变量：正方形的边长
side_length = 100

# 用循环画正方形（重复4次）
for i in range(4):
    t.forward(side_length)  # 前进 side_length 像素
    t.right(90)             # 右转 90 度

# 隐藏海龟，保持画面
t.hideturtle()

# 保持窗口不关闭
turtle.done()
```

**关键点：**
- `side_length = 100`：使用**变量**控制边长
- 修改 `side_length` 的值，就能画出不同大小的正方形！
- `for i in range(4)`：循环 4 次，画 4 条边

## M-TRIX百科
- `import turtle` = 召唤海龟：导入海龟绘图模块
  > 使用方法：`import turtle` 只需要写一次
- `t = turtle.Turtle()` = 创建海龟：生成一个海龟对象
  > 使用方法：`t = turtle.Turtle()` 给你的海龟起个名字
- `t.forward()` = 海龟前进：让海龟向前移动
  > 使用方法：`t.forward(100)` 前进 100 像素
- `t.right()` = 海龟右转：让海龟顺时针旋转
  > 使用方法：`t.right(90)` 右转 90 度
- `t.left()` = 海龟左转：让海龟逆时针旋转
  > 使用方法：`t.left(90)` 左转 90 度

## 创造与协作

### 基础任务：画你的领地
目标：用正方形画出你的 Minetest 领地边界！

```python
import turtle

t = turtle.Turtle()
t.speed(5)

# 定义领地大小
size = 200

# 画领地
for i in range(4):
    t.forward(size)
    t.right(90)

# 写领地名字
t.penup()
t.goto(0, -20)
t.pendown()
t.write("M-Boy的领地", align="center", font=("Arial", 16, "normal"))

t.hideturtle()
turtle.done()
```

### 进阶任务：画迷宫入口
目标：画一个复杂的迷宫入口，包含多个正方形！

```python
import turtle

t = turtle.Turtle()
t.speed(5)

# 画3个嵌套的正方形
sizes = [200, 150, 100]  # 三个不同的边长

for size in sizes:
    # 移动到起始位置
    t.penup()
    t.goto(-size/2, -size/2)
    t.pendown()
    
    # 画正方形
    for i in range(4):
        t.forward(size)
        t.right(90)

t.hideturtle()
turtle.done()
```
