### 知识点 0004 导入海龟库

#### Minetest 情境引入

你抵达了 Minetest 的神秘岛屿，发现了一只会画画的海龟。它告诉你，只有学会了它的语言，你才能让它帮你建造更复杂的建筑。

#### 核心知识点

导入库：在 Python 中，我们可以使用 `import` 语句来引入外部库，从而使用库中提供的功能。海龟库（Turtle）是一个内置的图形库，可以用来绘制各种图形。

别名：在导入库时，可以使用 `as` 关键字为库指定一个别名，这样在使用库中的功能时就可以用更简短的名字。

海龟对象：通过导入海龟库，我们可以用`turtle.Turtle()`或者`turtle.Pen()`创建一个海龟对象，这个对象就像一个画笔，可以在屏幕上移动并绘制图形。

#### 代码演示

导入方法一

```python
import turtle  # 导入海龟库

# 创建一个海龟对象
my_turtle1 = turtle.Turtle()
my_turtle2 = turtle.Pen()  # 也可以用 Pen() 来创建海龟对象
```

导入方法二


```python
import turtle as t  # 导入海龟库并指定别名为 t

# 创建一个海龟对象
my_turtle = t.Turtle()
```

### 知识点 0006 海龟对象的基本方法

#### Minetest 情境引入

你已经创建了一只海龟对象，现在你想让它帮你建造一座房子。为了让它按照你的要求工作，你需要了解它的基本属性和方法。

#### 核心知识点

海龟对象的基本方法：海龟对象有一些基本方法， `forward()`、`backward()`、`left()`、`right()` ，可以用来控制海龟的移动和转向。

#### 代码演示

绘制正方形：

```python
import turtle  # 导入海龟库
# 创建一个海龟对象
my_turtle = turtle.Turtle()

# 绘制正方形
my_turtle.forward(100)  # 向前移动 100 像素
my_turtle.left(90)     # 向左转 90 度
my_turtle.forward(100)  # 向前移动 100 像素
my_turtle.left(90)     # 向左转 90 度
my_turtle.forward(100)  # 向前移动 100 像素
my_turtle.left(90)     # 向左转 90 度
my_turtle.forward(100)  # 向前移动 100 像素
my_turtle.left(90)     # 向左转 90 度
```

绘制三角形：

```python
import turtle  # 导入海龟库

# 绘制三角形
```

#### 互动提问

你觉得这些方法是如何控制海龟的移动和转向的？如果你想让海龟画出一个五边形，你需要使用哪些方法呢？

#### 随堂笔记

海龟对象的基本方法包括 `forward()`、`backward()`、`left()`、`right()`，这些方法可以用来控制海龟的移动和转向。通过组合使用这些方法，我们可以让海龟绘制出各种图形。如果想让海龟画出一个五边形，我们需要使用 `forward()` 方法来控制海龟的前进，使用 `left()` 方法来控制海龟的转向，每次转向的角度应该是 72 度（360 度除以 5）。
