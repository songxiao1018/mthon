### 知识点 0001 输出字符串

#### Minetest 情境引入

你出生在Minetest的世界，现在需要和大家打招呼，你会怎么做呢？


#### 核心知识点

print() 函数：用于输出信息到控制台，可以输出字符串、数字、变量等。


#### 代码演示

```python
# 输出字符串

```


#### 互动提问

请问该如何输出完整的自我介绍呢？

```python

```

#### 随堂笔记

print() 函数是 Python 中最基本的输出函数，可以用来输出各种类型的数据，包括字符串、数字、变量等。使用 print() 函数可以让我们在控制台上看到程序的输出结果。

print() 函数的语法非常简单，只需要将要输出的内容放在括号内即可。我们可以使用引号（单引号或双引号）来定义字符串，也可以直接输出数字或变量。



### 知识点 0002 变量与变量赋值

#### Minetest 情境引入

你在Minetest的世界里发现了一些宝藏，你想把它们记录下来，以便以后使用，你会怎么做呢？

#### 核心知识点

变量：用于存储数据的命名空间，可以存储不同类型的数据，如字符串、数字、列表等。
变量赋值：使用等号（=）将一个值赋给一个变量。
变量的命名规则：变量名必须以字母或下划线开头，后面可以跟字母、数字或下划线。变量名区分大小写，不能使用 Python 的保留字作为变量名。
变量的使用：我们可以使用变量来存储用户输入的数据、计算结果或者任何需要在程序中使用的信息。通过变量，我们可以让程序更加灵活和动态，能够处理不同的数据和情况。

#### 代码演示

```python

# 定义变量并赋值
name = "M-Boy"
age = 30

# 输出变量
print("Name:", name)
print("Age:", age)

# 更新变量的值
age = 31
print("Updated Age:", age)
```

#### 互动提问

当你在地上捡起了一个宝藏，你会如何记录它的名称和数量呢？

```python
# 记录宝藏的名称和数量
treasure_name = "Diamond"
treasure_quantity = 5
treasure_num = 1

print("Treasure Name:", treasure_name)
print("Treasure Quantity:", treasure_quantity)
print("Treasure Number:", treasure_num)

# 又发现一个新的，需要更新宝藏的数量
treasure_num = 2
print("Updated Treasure Number:", treasure_num)
```

#### 随堂笔记

变量是编程中非常重要的概念，它们允许我们存储和操作数据。通过变量赋值，我们可以将不同类型的数据存储在变量中，并且可以随时更新它们的值。使用变量可以让我们的代码更加灵活和易于维护。
变量的命名规则：变量名必须以字母或下划线开头，后面可以跟字母、数字或下划线。变量名区分大小写，不能使用 Python 的保留字作为变量名。
变量的使用：我们可以使用变量来存储用户输入的数据、计算结果或者任何需要在程序中使用的信息。通过变量，我们可以让程序更加灵活和动态，能够处理不同的数据和情况。



### 知识点 0003 注释

#### Minetest 情境引入

你在Minetest的世界里建造了一个复杂的建筑，你想让其他人也能理解你的设计思路，你会怎么做呢？

#### 核心知识点

注释：用于在代码中添加说明性文字，帮助自己和他人理解代码的功能和逻辑。注释不会被程序执行。

单行注释：使用井号（#）开头，后面跟随注释内容。

多行注释：使用三引号（''' 或 """）包裹起来，可以用于注释多行内容。

#### 代码演示

```python
# 这是一个单行注释，解释下面的代码
print("Hello, World!")  # 这也是一个单行注释，解释这一行代码

'''
这是一个多行注释，可以用于解释多行代码
或者提供更详细的说明
'''
print("Welcome to Minetest!")
```

#### 互动提问

当你在地上建造了一个复杂的建筑，你会如何使用注释来解释你的设计思路呢？

```python
# 这是一个单行注释，解释下面的代码
print("Hello, World!")  # 这也是一个单行注释，解释这一行代码

'''
这是一个多行注释，可以用于解释多行代码
或者提供更详细的说明
'''
print("Welcome to Minetest!")
```

#### 随堂笔记

注释是编程中非常重要的工具，它们可以帮助我们解释代码的功能和逻辑，使代码更易于理解和维护。通过使用单行注释和多行注释，我们可以在代码中添加详细的说明，帮助自己和他人更好地理解代码的意图和实现细节。
