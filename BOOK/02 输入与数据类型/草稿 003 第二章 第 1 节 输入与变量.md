### 知识点 0007 整型变量

#### Minetest 情境引入

你在 Minetest 的编程世界中继续冒险，有一天你突然发现村庄里有一堆的干草块，你想要收集并记录他们，你该用什么样的变量来存储这些干草块的数量呢？

#### 核心知识点

- **整数变量 (Integer Variable)** = 数字容器：一个用来存储整数（没有小数的数字）的“盒子”，你可以在程序中使用它来保存和操作数字。
  > 使用方法：`变量名 = 整数值`，例如 `score = 10`。
  > 示例：`score = 10` 意思是“我创建了一个叫score的变量，并把数字10放进这个变量里”。

#### 代码演示

```python
# 定义一个整数变量并赋值
score = 10
print(score)  # 输出变量的值
```

#### 互动提问

请问如何定义一个变量来存储你的年龄，并输出它呢？

```python
# 定义一个整数变量来存储年龄
age = 25
print(age)  # 输出变量的值
```

#### 随堂笔记

整数变量是 Python 中用来存储整数的基本数据类型。它们可以用来表示各种数量、计数、分数等。定义一个整数变量非常简单，只需要使用等号（=）将一个整数值赋给一个变量名即可。通过使用整数变量，我们可以在程序中进行各种数学运算和逻辑判断，使我们的程序更加灵活和功能丰富。

### 知识点 0010 输入方法 input()

#### Minetest 情境引入

有一天你来到朋友家里，发现他家桌子上明晃晃摆着一个大密码箱，朋友说需要输入密码才能打开这个箱子。朋友信心满满地说：“这个密码箱的密码是你最喜欢的颜色，你能猜到吗？”你想了想，决定输入你的最喜欢的颜色来试试，你该用什么样的方法来输入这个信息呢？

#### 核心知识点

- **输入方法 input()** = 输入工具：一个用来获取用户输入的函数，你可以在程序中使用它来让用户输入信息。
  > 使用方法：`变量名 = input("提示信息")`，例如 `name = input("请输入你的名字: ")`。
  > 示例：`name = input("请输入你的名字: ")` 意思是“我创建了一个叫name的变量，并使用input()函数提示用户输入他们的名字，然后把用户输入的值存储在name变量里”。

#### 代码演示

```python
# 使用 input() 函数获取用户输入
name = input("请输入你的名字: ")
print("你好，" + name + "！")  # 输出欢迎信息
```

#### 互动提问

请问如何使用 input() 函数来获取用户的年龄，并输出一个欢迎信息呢？

```python
# 获取用户输入的年龄
age = input("请输入你的年龄: ")
print("你输入的年龄是: " + age)  # 输出用户输入的年龄
```

#### 随堂笔记

input() 函数是 Python 中用来获取用户输入的基本函数。它会暂停程序的执行，等待用户输入信息，并将用户输入的内容作为字符串返回。我们可以将这个返回值赋给一个变量，以便在程序中使用。通过使用 input() 函数，我们可以让我们的程序与用户进行交互，获取用户的输入信息，使我们的程序更加动态和有趣。

### 知识点 0011 数据类型

#### Minetest 情境引入

你发现输入的input好像只能输入文本信息，但是你想要输入数字来进行计算，你该怎么办呢？

#### 核心知识点

- **数据类型 (Data Type)** = 数据的种类：不同类型的数据有不同的特征和用途，常见的数据类型包括整数（int）、浮点数（float）和字符串（str）。
  > 整数（int）：用于表示没有小数的数字，例如 `score = 10`。
  > 浮点数（float）：用于表示带有小数的数字，例如 `pi = 3.14`。
  > 字符串（str）：用于表示文本信息，例如 `name = "Alice"`。

- **数据类型转换** = 将一种数据类型转换为另一种数据类型的方法，例如使用 `int()` 将字符串转换为整数，使用 `float()` 将字符串转换为浮点数。
  > 示例：`age = int(input("请输入你的年龄: "))` 意思是“我使用input()函数获取用户输入的年龄，并使用int()函数将输入的字符串转换为整数，然后把这个整数存储在age变量里”。

#### 代码演示

```python
# 获取用户输入的年龄并转换为整数
age = int(input("请输入你的年龄: "))
print("你输入的年龄是: " + str(age))  # 输出用户输入的年龄
```

```python
# 获取用户输入出生年份并计算年龄
birth_year = int(input("请输入你的出生年份: "))
current_year = 2024
age = current_year - birth_year
print("你今年的年龄是: " + str(age))  # 输出计算出的年龄
```

#### 互动提问

请问如何使用数据类型转换来获取用户输入的身高，并计算出他的BMI呢？

```python
# 获取用户输入的身高并转换为浮点数
height = float(input("请输入你的身高（米）: "))
weight = float(input("请输入你的体重（千克）: "))
bmi = weight / (height * height)  # 计算BMI
print("你的BMI是: " + str(bmi))  # 输出计算出的BMI
```

#### 随堂笔记

数据类型是 Python 中用来表示不同种类数据的分类。常见的数据类型包括整数（int）、浮点数（float）和字符串（str）。每种数据类型都有其特定的用途和特点。我们可以使用数据类型转换函数来将一种数据类型转换为另一种数据类型，以便在程序中进行不同的操作和计算。通过理解和使用不同的数据类型，我们可以让我们的程序更加灵活和功能丰富。
