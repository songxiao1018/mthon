### Multi-TRIX百科
介绍本节课的核心知识点，它们是你与编程世界对话的基石。

- **整数变量 (Integer)** = 数字容器：【**复习**第 1 课】一个专门用来存放整数（没有小数部分的数字）的坚固盒子，比如用来记录你收集的方块数量、你的生命值。
  > 使用方法：`变量名 = 整数值`，例如 `hay_blocks = 10`。

- **浮点型变量 (Float)** = 精密容器：一个能存放带小数点的数字的精密容器，适合存储需要精确测量的值，比如距离、时间、比例。
  > 使用方法：`变量名 = 小数值`，例如 `river_width = 5.6`。

- **字符串变量 (String)** = 文字胶囊：一个能封装任何文本信息（文字、符号、数字串）的胶囊，用来存放名字、谜语、对话。
  > 使用方法：`变量名 = "一串文字"`，例如 `riddle = “我是谁？”`。

- **输入方法 input()** = 信息收集器：一个能暂停世界，等待你从键盘输入信息，并把信息带回来的神奇工具。它带回来的信息总是“字符串”类型的。
  > 使用方法：`收到的信息 = input(“给你的提示：”)`，例如 `name = input(“请输入你的名字: ”)`。

- **数据类型转换** = 万能变形术：一种能将一种类型的数据（特别是字符串）转换成另一种类型的魔法。常用咒语是 `int()`（变整数）、`float()`（变小数）、`str()`（变文本）。
  > 使用方法：`目标类型(数据)`，例如 `age = int(“12”)` 会把字符串“12”变成整数12。

## 2.2 Bug猎人 老师报错了！你能帮老师修复代码吗？

程序世界有时会出现一些小“Bug”（错误）。别担心，读懂错误信息是成为编程高手的第一步。来帮老师修复下面的代码吧！

**常见错误类型：**
1.  **NameError (名称错误)**：使用了没有定义过的变量名。
2.  **TypeError (类型错误)**：对数据进行了不适合其类型的操作，比如用字符串做数学计算。
3.  **SyntaxError (语法错误)**：代码的写法不符合Python规则，比如少了引号、括号。
4.  **ValueError (值错误)**：给函数传递了它无法处理的值，比如把“abc”传给`int()`。

### Bug 1: NameError - 调用了不存在的“伙伴”
```python
# 错误代码
myGold = 50
print(“我拥有的金锭是：”, myGld) # 老师不小心拼错了变量名！
```
**报错信息**：`NameError: name 'myGld' is not defined`
**诊断**：我们告诉`print`去打印一个叫`myGld`的变量，但我们只定义了`myGold`。Python找不到`myGld`，所以报错。
**修复方案**：将`myGld`改成正确的变量名`myGold`。

### Bug 2: TypeError - 让文字和数字“打架”
```python
# 错误代码
age_from_input = input(“你今年多大了？ ”)
years_later = age_from_input + 10 # 试图将字符串和数字相加
print(“10年后你就”, years_later, “岁了。”)
```
**报错信息**：`TypeError: can only concatenate str (not “int“) to str`
**诊断**：`input()`得到的是字符串（比如“12”），而`10`是整数。Python不知道你是想把“12”和10连接成“1210”，还是想把它们相加成22。
**修复方案**：需要先将字符串转换成整数，再进行计算。
```python
age_from_input = input(“你今年多大了？ ”)
age_int = int(age_from_input) # 关键转换！
years_later = age_int + 10
print(“10年后你就”, years_later, “岁了。”)
```

### Bug 3: SyntaxError - 忘记关上的“话匣子”
```python
# 错误代码
secret_message = “这是一个未完成的秘密 # 字符串的引号没有成对出现
print(secret_message)
```
**报错信息**：`SyntaxError: EOL while scanning string literal`
**诊断**：Python在读取一行代码时，直到行尾（EOL）都没找到与开头引号配对的另一个引号，语法不完整。
**修复方案**：在字符串末尾加上匹配的引号。`secret_message = “这是一个未完成的秘密”`
