> Multi-Boy 整理完魔法背包的第二天，脑子里的"三板斧"还亮着吗？我们用 5 分钟把上一节最常用的 5 个操作磨一磨，免得一会儿切新宝物时手生。

### 关键操作速查

| 类别 | 操作 | 一句话回忆 |
| :--- | :--- | :--- |
| **加** | `bag.append(x)` | 在背包**末尾**追加一件物品 |
| **加** | `bag.insert(i, x)` | 插队到第 `i` 格 |
| **删** | `bag.remove(x)` | 按**物品名**删除（删第一个匹配项） |
| **删** | `bag.pop(i)` | 按**位置**拿出并返回 |
| **删** | `del bag[i]` | 按**位置**直接丢掉 |
| **改** | `bag[i] = new_val` | 用新物品替换第 `i` 格 |
| **取** | `bag[i]` / `bag[-1]` | 取第 `i` 个 / 最后一个 |

> 💡 口诀：**append 加结尾、insert 按位插、remove 按值删、pop 拿出还、del 直接丢**。

### 第一次热身

```python
# 复习 0015：增 / 删 / 改 各一招
bag = ['面包', '火把']
bag.append('水壶')            # 增：末尾加水壶
print(bag)                    # ['面包', '火把', '水壶']
bag[0] = '金苹果'              # 改：把面包升级
print(bag)                    # ['金苹果', '火把', '水壶']
del bag[1]                    # 删：扔掉火把
print(bag)                    # ['金苹果', '水壶']
```

2. **"把你第 2 格到第 5 格的宝物拿出来给我看看。"** —— 你要一次掏出**一整段**！
3. **"把整个清单倒过来念一遍，我要从最近的收获听起。"** —— 你要把列表**翻转**！

单格取物你已经会了（`backpack[2]`），但要**数数量**、**切一段**、**整个倒序**，这就要请出本节课的两大新魔法——**列表截取（切片）**和 **`len()` 物品计数器**。

### Multi-TRIX百科

本节课，我们将学习"魔法背包"的两件新工具。

- **列表截取（切片 Slice）** = 抽屉切片刀：用 `[头下标 : 尾下标 : 步长]` 的"刀法"，从背包里一刀切出一段连续物品。
  > 使用方法：`列表[头下标 : 尾下标 : 步长]`，**包头不包尾**（左闭右开）。
  > 示例：`loot[1:4]` 表示"从索引 1 开始，到索引 4 之前停下"，切出索引 1、2、3 三个元素。

- **切片省略写法**：
  > `列表[:尾下标]`：从头切到 `尾下标` 之前。
  > `列表[头下标:]`：从 `头下标` 切到末尾。
  > `列表[:]`：复制整个列表。
  > 示例：`loot[:3]` 等价于 `loot[0:3]`，`loot[2:]` 等价于 `loot[2:len(loot)]`。

- **步长（step）** = 跳着取物品：每跨 `步长` 个格子取一个。
  > 使用方法：`列表[头下标 : 尾下标 : 步长]`
  > 示例：`loot[::2]` 隔一个取一个（取偶数位）；`loot[::-1]` **倒序**取整个列表。

- **`len()` 函数** = 物品计数器：把列表整个放进去，立刻告诉你里面有几个元素。
  > 使用方法：`len(列表名)`，返回值为整数。
  > 示例：`len(['面包', '火把', '水壶'])` → `3`。

- **嵌套列表的长度**：嵌套的子列表**整体算 1 个元素**，不会被拆开数。嵌套的列表相当于盒子里放了一个收纳袋将零散的物品集合在一起了。
  > 示例：`len([[1, 2], [3, 4, 5], 6])` → `3`（三个"格子"，里面分别装着 `[1,2]`、`[3,4,5]`、`6`）。

> 💡 红石小技巧：切片 `lst[a:b]` 和 `range(a, b)` 的区间规则完全一致——**左闭右开**，含头不含尾。复习 `range` 时可以对照记忆。

### 第一次运行

让我们通过几个完整的代码例子，亲手体验"切片"和"长度"的威力吧！

```python
# === 示例 1：物品计数器 len() ===
loot = ['面包', '火把', '水壶', '地图', '金锭', '钻石', '末影珍珠']
print('一共', len(loot), '件宝物')              # 7
print('最后一件是：', loot[len(loot) - 1])       # 末影珍珠，等价于 loot[-1]

# 嵌套列表：子列表整体算 1 格
chests = [['钻石', '金锭'], ['火把'], ['面包', '水壶', '地图']]
print('共有', len(chests), '个宝箱')             # 3
print('第一个宝箱里有', len(chests[0]), '种宝物')  # 2
```

```python
# === 示例 2：抽屉切片刀 [头:尾:步] ===
loot = ['面包', '火把', '水壶', '地图', '金锭', '钻石', '末影珍珠']
print('第 2 到 4 件：', loot[1:4])   # ['火把', '水壶', '地图']
print('前 3 件：', loot[:3])          # ['面包', '火把', '水壶']
print('从第 5 件到末尾：', loot[4:])  # ['金锭', '钻石', '末影珍珠']
print('完整倒序：', loot[::-1])       # ['末影珍珠', '钻石', '金锭', '地图', '水壶', '火把', '面包']
print('奇数位：', loot[::2])          # ['面包', '水壶', '金锭', '末影珍珠']
```

```python
# === 示例 3：联合使用——给村长的一份战利品报告 ===
loot = ['面包', '火把', '水壶', '地图', '金锭', '钻石', '末影珍珠']
```

## 5.2 Bug猎人：老师报错了！你能帮老师修复代码吗？

> 村长让 Multi-Boy 写报告，可 Multi-Boy 写错了好几处。下面的代码就是他在冒险日志里写的"残次品"，请你当一回 Bug 侦探，帮他一一修复！

**常见错误类型总结：**

1. **IndexError（索引错误）**：访问不存在的索引位置。`len(列表)` 是合法索引的最大值 +1，所以 `列表[len(列表)]` 会越界。
2. **TypeError（类型错误）**：对不支持切片的类型用了切片，如 `123[1:3]`。
3. **ValueError（值错误）**：用 `remove()` 删除列表中不存在的值。
4. **SyntaxError（语法错误）**：步长为 0 会导致语法/逻辑错误，如 `loot[::0]`。

### Bug 1：错把数字当列表

```python
# 老师想看自己出生年份的"后两位"
year = 2024
print(year[2:4])   # 报错：TypeError: 'int' object is not subscriptable
```

**报错信息**：`TypeError: 'int' object is not subscriptable`

**你的修复：**
```python
# 整数不能直接切片，要先转成字符串
year = 2024
year_str = str(year)              # 把整数变成字符串 '2024'
print(year_str[2:4])              # 输出：24
```

### Bug 2：把"长度"当成了最后一个索引

```python
# 老师想拿背包的"最后一件"
loot = ['面包', '火把', '水壶']
print(loot[len(loot)])            # 报错：IndexError: list index out of range
```

**报错信息**：`IndexError: list index out of range`

**你的修复：**
```python
# len() 返回的是"元素个数"，最大合法索引是 len-1
loot = ['面包', '火把', '水壶']
print(loot[len(loot) - 1])        # 输出：水壶
# 更简单的写法：直接用反向索引
print(loot[-1])                   # 输出：水壶
```

### Bug 3：步长写成了 0

```python
# 老师想"每 0 个取一个"——结果 Python 懵了
loot = ['面包', '火把', '水壶', '地图']
print(loot[::0])                  # 报错：ValueError: slice step cannot be zero
```

**报错信息**：`ValueError: slice step cannot be zero`

**你的修复：**
```python
# 步长不能为 0，要么写 1（每个都取）要么写其他正/负整数
loot = ['面包', '火把', '水壶', '地图']
print(loot[::1])                  # 输出：['面包', '火把', '水壶', '地图']  ← 等于原列表
print(loot[::2])                  # 输出：['面包', '水壶'] ← 隔一个取一个
```

### Bug 4：`remove()` 删了一个不存在的怪物

```python
# 老师想从战利品中扔掉"末影龙"
loot = ['面包', '火把', '水壶', '地图']
loot.remove('末影龙')              # 报错：ValueError: list.remove(x): x not in list
print(loot)
```

**报错信息**：`ValueError: list.remove(x): x not in list`

**你的修复：**
```python
# 删除前先用 in 检查一下，避免 ValueError
loot = ['面包', '火把', '水壶', '地图']
if '末影龙' in loot:                # 先问问：列表里有没有这个怪物？
    loot.remove('末影龙')
else:
    print('背包里没有末影龙!')
# 正确做法：删一个真实存在的物品
loot.remove('水壶')
print(loot)                        # 输出：['面包', '火把', '地图']
```
