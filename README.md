## 🎮 项目简介
本项目是一套**面向青少年的 Python 入门到进阶互动式教学手册**，以沙盒游戏 **Minetest** 为沉浸式背景，通过主角 **Multi-Boy**、**Multi-Girl** 的冒险故事串联知识点。
手册严格对齐 **CCF GESP Python 1–4 级考纲**，采用 **“纯逻辑编程 + Turtle 可视化”双轨教学法**，让孩子在“玩游戏”的过程中自然掌握编程思维。

**项目周期**：3–6 个月  
**开发模式**：单人独立开发  
**当前进度**：🔄 项目重启中 —— 仅保留 5 个 T-XX 模板，等待新版知识点体系落地后重新生成课件

## 🧩 核心设计理念
### 1️⃣ 世界观驱动（Worldview-Driven）
- **项目含义**：本项目是基于 **Minetest** 沙盒世界的**综合矩阵教学手册**。
  - **Multi-TRIX** 取”多元（Multi）× 矩阵（TRIX，matrix 词根）”之意，承载”在 Minetest 开放世界中看清代码背后的逻辑结构”的愿景。
  - **THON** 是本手册的四步教学理念：**T**hink（思考）→ **H**ack（创造）→ **O**ptimize（优化）→ **N**ext（进阶），对应青少年从认知到创造的完整学习循环。
- 所有知识点都包裹在”生存建造”、”机关设计”、”自动化农场”等游戏任务中。

### 2️⃣ 单课单一职责（Single-Purpose Lesson）
每节课只做一件事，拒绝混杂：
| 课型     | 数量  | 作用                     |
| -------- | ----- | ------------------------ |
| 📘 知识课 | 21 节 | 讲清语法、逻辑、原理     |
| 🐢 海龟课 | 17 节 | 用 Turtle 把代码“画出来” |
| 📝 考级课 | 13 节 | 刷 GESP 真题、避坑训练   |
| 🔄 复习课 | 穿插  | 综合项目实战             |

### 3️⃣ 双轨输出验证
- **逻辑验证**：IDLE 中运行代码，解决生存问题（计算资源、判断陷阱）
- **视觉验证**：Turtle 实时绘图，让抽象逻辑“看得见”

## 🎨 风格速览
每个知识点 = **一句话比喻 + 可运行代码 + 行内注释**：

diamonds = 3            # 背包存钻石

## 🛠️ 使用指南
### 适合谁用？
- 10–16 岁零基础青少年
- 准备参加 **GESP Python 考级** 的学生
- 想用游戏化方式教编程的老师 / 家长

### 如何使用？
1. 按顺序学习，**不要跳课**
2. 每节课先读故事 → 敲代码 → 完成任务
3. 海龟课务必动手画图
4. 考级课限时训练
